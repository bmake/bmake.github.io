Studiengangsuche mit EduGraph

Fall:
Studienführer versprechen Unterstützung bei der Studiensuche. Die Daten müssen von den Hochschulen in ca. 50 Systemen gepflegt werden. Die Hochschul-Webseiten halten diese Daten ohnehin vor. Eine einheitliche Auszeichnung k&ouuml;nnte zu erheblichen Einsparungen führen.

Lösung:
Studieninteressierte greifen über eine App auf die föderierten Daten der Studiengänge einer Fachdomäne zu. Die Daten werden dezentral von den Verantwortlichen an den Hochschulen gepflegt. Fachliche Schwerpunkte und Qualifikationsprofile werden transparent.

  • Wissensschema
  • Interaktives Architekturmodell
  • Projektressourcen auf GitHub

Erkenntnisse aus dem Projekt:
Eine semantische Auszeichnung Web-Content, der im CMS strukturiert abgelegt ist, kann über einfache Ergänzungen in den Seitentemplates erreicht werden. Erfahrungen wurden auch bei der semantischen Textanalyse von Modulkatalogen gesammelt, die im neuen Projekt Digitale Modulkataloge von Nutzen sind.
